# To-Do 리스트 웹앱

바닐라 JavaScript로 만든 심플한 할 일 관리 웹 애플리케이션입니다.

## 주요 기능

- **할 일 추가** — 텍스트 입력 후 버튼 클릭 또는 Enter 키로 추가
- **중복 방지** — 동일한 내용의 할 일은 등록 불가
- **완료 체크** — 체크박스로 완료/미완료 토글, 완료 시 취소선 표시
- **삭제** — 개별 항목 삭제 및 완료된 항목 일괄 삭제
- **필터** — 전체 / 진행중 / 완료 필터로 목록 조회
- **통계** — 전체·완료·미완료 개수 실시간 표시
- **날짜 표시** — 추가일 및 완료일 표시
- **데이터 유지** — LocalStorage에 저장되어 새로고침 후에도 데이터 보존

## 기술 스택

| 구분 | 기술 |
|------|------|
| 마크업 | HTML5 |
| 스타일 | CSS3 |
| 로직 | Vanilla JavaScript (ES6+) |
| 저장소 | LocalStorage |

## 파일 구조

```
ToDo_complete/
├── index.html   # 앱 구조 (HTML)
├── style.css    # 스타일시트
├── main.js      # 앱 로직 (JS)
└── README.md    # 프로젝트 설명
```

## 실행 방법

별도의 설치나 빌드 과정 없이 `index.html` 파일을 브라우저에서 열면 바로 실행됩니다.
